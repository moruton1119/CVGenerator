# Resume Helper (履歴書作成支援ツール)

ブラウザ上で動作する、シンプルな履歴書情報収集ツールです。
入力したデータは自動的にブラウザに保存され、後で再開したり、JSONファイルとしてバックアップ保存することができます。

## 使い方

1. `index.html` をブラウザ（Chrome, Edge, Firefoxなど）で開きます。
2. フォームに情報を入力します。
   - **基本情報**: 氏名、連絡先など
   - **職務経歴**: 「+ 追加」ボタンで経歴を自由に追加できます
   - **学歴**: 「+ 追加」ボタンで学歴を追加できます
3. 入力内容は自動的に保存されます（Local Storage）。
4. **保存/読み込み**:
   - `JSON保存`: 入力内容をファイルとしてダウンロードします。
   - `JSON読込`: 保存したファイルを読み込んで復元します。
5. **印刷/PDF化**:
   - `PDF保存 / 印刷` ボタンを押すと、印刷プレビューが開きます。
   - ブラウザの機能で「PDFとして保存」を選択して出力してください。

## ファイル構成

- `index.html`: アプリケーション本体
- `style.css`: スタイルシート（印刷用スタイル含む）
- `script.js`: アプリケーションの動作ロジック
